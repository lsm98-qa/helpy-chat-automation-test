# Changelog

이 문서는 프로젝트의 주요 변경 사항을 기록합니다.

## [0.2.0] - 2026-04-07

### 추가
- **채팅(chat) 시나리오 확장:**
  - 대화 입력 후 AI 모델 전환 시 채팅 일관성 검증 시나리오 추가
  - 검색창 결과 개수 검증 및 21개 이상 데이터 세팅 시나리오 추가
  - 이름 변경 시 빈 값 입력 검증, 창 가로 축소 시 기능 동작 검증 시나리오 추가
- **에이전트(agent) 시나리오 확장:**
  - 검색 결과 없음(empty state) UI 및 에이전트 만들기 이동 검증 추가
  - 내 에이전트 게시 상태 기반 페이지 이동 검증 추가
  - 게시된 에이전트 이름 수정 검증 추가
- **도구(tools) 시나리오 확장:**
  - 수업지도안 빠른 생성/정교한 생성 시나리오 추가
  - 행동특성 및 종합의견, 세부 특기사항에서 학생 추가 후 생성 검증 시나리오 추가
  - 심층 조사 테스트를 단계별 시나리오로 분리
- **운영/자동화:**
  - `.gitlab-ci.yml` 추가 및 수동 실행 설정 반영

### 변경
- **공통 로깅 체계:**
  - `conftest.py`에 AAA(`Arrange-Act-Assert`) 공통 로깅 픽스처(`testlog`) 추가
  - auth/chat/agent/tools 테스트 전반에 AAA 로그 라인 적용
- **테스트 실행 설정:**
  - `pytest.ini`에서 xfail reason 출력(`-rxX`) 가시성 강화
  - 외부 라이브러리 로그 레벨 조정으로 로그 노이즈 감소
  - WebDriver `implicit wait` 비활성화로 명시적 대기 중심으로 정리
- **문서/의존성:**
  - `requirements.txt`에 문서/이미지 처리 관련 의존성 업데이트 반영

### 수정
- **채팅 검색 안정화:**
  - 대소문자 무시 비교, 공백 포함 제목 처리, 최대 비교 개수(상위 20개) 검증 로직 보완
  - 검색 결과 로딩 대기 중 예외 처리 추가
- **에이전트 안정화:**
  - 에이전트 목록 스크롤 탐색 및 저장 완료 카드 탐색 로직 개선
  - stale element 및 저장 버튼 관련 flaky 이슈 완화
- **도구 시나리오 안정화:**
  - 주관식 퀴즈 생성 시 이전 객관식 결과 잔존 문제 수정
  - 수업지도안/심층 조사 시나리오 분기 및 assert 로직 보완

### 호환성
- **Breaking Change:** 없음
